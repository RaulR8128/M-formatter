# 📊 M-Formatter: Personalización de Lenguaje M

**M-Formatter** es una herramienta web diseñada para analistas de datos que buscan llevar el orden de su código en lenguaje M (Power Query) al siguiente nivel. Diseñada por **Raúl Ramos**, especialmente para la comunidad hispana.

---

## 🚀 ¿Por qué M-Formatter?

El código generado por Power Query suele ser difícil de leer. **M-Formatter** permite auditar y transformar la estructura visual según reglas de estilo profesionales.

### ✨ Características Principales
* **Interfaz Inspirada en Power Query:** Flujo de trabajo familiar para usuarios de Power BI y Excel.
* **Validación de Sintaxis:** Filtro de seguridad que asegura que el código sea válido antes de aplicar cambios.
* **Personalización de Indentación:** Control total sobre espacios (2, 4 o Tabs).
* **Refactorización de Nombres:** Conversión automática de pasos a `CamelCase` o `snake_case`.
* **Auditoría de Consulta:** Panel lateral con conteo de pasos y nivel de complejidad.

---

## 🛠️ Cómo Funciona

1. **Pegar:** Introduce tu código crudo en el editor izquierdo.
2. **Analizar:** El sistema valida la estructura y extrae los pasos aplicados.
3. **Configurar:** Ajusta las preferencias de estilo en el menú superior.
4. **Aplicar:** Genera el código limpio y cópialo de vuelta a Power BI o Excel.

---

## 🔒 Privacidad y Seguridad

**Tu información es privada.** M-Formatter procesa todo el código localmente en tu navegador. **No enviamos tus datos ni rutas de archivos a ningún servidor externo.**
